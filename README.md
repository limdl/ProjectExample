项目示例：
1、分析挖掘
	a、spss modeler挖掘潜在客户
	sql语句加工出数据宽表，用spss modeler中的logistics回归挖掘潜在客户
	
	b、瑞幸微博评论文本情感分析
	对每条微博评论，量化到0-1直接的得分，越接近1越好，分析社交媒体对瑞幸的评价

2、数据产品
	a、系统实现“销售目标分解系统需求”的示例文档

3、数据可视化
	a、excel+SQL可视化图表
	excel中用odbc连接sql，整理出数据，做出可视化图表，完成分析
	
	b、python+django可视化图表
	django实现日常需查看可视化图表，共享到局域网内用户都能直接浏览器访问地址查看

4、通用爬虫
	a、python通用爬虫
	generalspider.py为通用爬虫类，含有基本类、调用selenium类、json解析类，其他.py继承generalspider.py、重构函数实现任一网站来源的快速爬取
	
	b、scrapy通用爬虫
	scrapy框架下实现通用爬虫示例
	
	c、八爪鱼
	用八爪鱼实现爬虫示例，八爪鱼在一些“小数据”可以快速配置出爬虫规则，在“小数据”上比程序采集有速度上的优势
